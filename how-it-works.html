How it works

From signup to sideline in minutes.

1. Sign up

Coach creates an account with email and password.

2. Create team

Dashboard creates a team workspace in Supabase.

3. Open predictor

The app is protected behind login.

4. Coach the game

Use live prediction, reports, and sideline mode.

Supabase setup steps

  1. Create a free Supabase project.
  2. Copy your Project URL and anon public key into assets/config.js.
  3. Run the SQL in supabase_schema.sql inside Supabase SQL Editor.
  4. In Authentication settings, set your Site URL to your Netlify URL.
  5. Deploy the folder to Netlify.
  6. Test signup, login, dashboard, and opening the app.

The exact SQL and step-by-step launch checklist are included in README_SUPABASE_LAUNCH.md.
